# Python countdown timer

import time

my_time = int(input("Enter the time in seconds: "))

for x in range(my_time, 0, -1):
    seconds = x % 60
    minutes = int(x / 60) % 60
    hours = int(x / 3600)

    print(f"{hours:02}:{minutes:02}:{seconds:02}")
    time.sleep(1)

print("TIME'S UP!")
    

"""
Notes (Step-by-Step Explanation):

- The program creates a countdown timer based on user input in seconds.
- input() is used to take time from the user; int() converts it into a whole number.
- range(my_time, 0, -1) creates a loop that counts down from the entered time to 1.
- seconds are calculated using modulus (x % 60).
- minutes are derived by dividing by 60 and taking modulus 60.
- hours are calculated by dividing by 3600 (total seconds in an hour).
- f-string formatting {hours:02}:{minutes:02}:{seconds:02} ensures two-digit display (e.g., 01:05:09).
- time.sleep(1) pauses the program for 1 second between each loop iteration.
- Once the countdown ends, "TIME'S UP!" is displayed.
- Demonstrates key concepts: loops, time module usage, arithmetic operations, and formatted output.
"""
