# Python weight converter

weight = float(input("Enter your weight: "))
unit = input("Kilograms or Pounds? (K or L): ")

if unit == "K":
    weight = weight * 2.205
    unit = "Lbs."
    print(f"Your weight is: {round(weight, 1)} {unit}")

elif unit == "L":
    weight = weight / 2.205
    unit = "Kgs."
    print(f"Your weight is: {round(weight, 1)} {unit}")

else:
    print(f"{unit} was not valid")


"""
Notes (Step-by-Step Explanation):

- input() is used to take user input; float() converts weight into a numeric value for calculations.
- The user selects the unit: 'K' for Kilograms or 'L' for Pounds.
- If 'K' is entered, the weight is converted to pounds using weight * 2.205.
- If 'L' is entered, the weight is converted to kilograms using weight / 2.205.
- round(weight, 1) is used to limit the result to 1 decimal place for cleaner output.
- f-strings are used to format and display the result clearly.
- The else block handles invalid inputs and shows an appropriate message.
- Demonstrates key concepts: input handling, conditional logic, arithmetic operations, and formatting.
"""
